Projet : Plateforme de personnalisation d’histoires « enfants + adultes »
Objectif : Permettre aux parents, éducateurs et adultes d’offrir ou créer des histoires personnalisées (nom, âge, photo, image personnalisée) au format PDF, couvrant deux univers (kids & adults).
Technologies : Backend NestJS + MongoDB + JWT | Frontend Next.js 14 + TypeScript + TailwindCSS + shadcn/ui + Framer Motion | PDF traitement via pdfjs-dist + pdf-lib | Cartoonification image via API externe.
Rôles : admin (gère templates), user (personnalise, génère PDF).
Valeur : Réutilisation de templates dynamiques, personnalisation immédiate, impression possible, mémoire émotionnelle.
Marché principal : Tunisie (français/arabe), extension internationale possible.
